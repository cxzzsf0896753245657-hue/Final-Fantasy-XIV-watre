📋 2026/07/15
  • 右側新增三個功能分頁：🧭 路線規劃、⚙️ 配件規劃、🔨 配件製作（均為 Beta）
  • 【路線規劃】依船艇等級、時間限制、目標道具自動搜尋最佳航線；支援地圖／地點篩選、強制包含或排除特定站點；結果依 EXP/分 排序，顯示預估回港時間、各 Tier 掉落道具與探索門檻
  • 【配件規劃】輸入目標道具，分析各探索站的探索值門檻差距，以顏色標示目前性能是否達標
  • 【配件製作】選擇目標配件與製作數量，自動計算成品素材、半成品、基礎素材所需數量；支援手動輸入已持有庫存，即時顯示缺少量；完成項目可打勾標記
  • 路線規劃與配件規劃中，對應「海圖跑點」未開航點的地點以 🔒 上鎖符號標示，不影響推薦結果但可清楚辨識
  • 素材表格四欄改為可橫向捲動的佈局，各欄設最小寬度，確保大數量時仍可完整顯示；隱藏捲軸 UI 但保留捲動功能
  • 配件欄目標數量由「－ 數字 ＋」按鈕組改為數字輸入框，支援直接輸入與上下箭頭調整
  • 左側面板縮窄至 460px，讓右側規劃區有更多顯示空間
  • 開發日記加入固定高度捲軸，避免版面持續延伸
📋 2026/07/10
  • 「海圖」區塊新增跑點記錄：每張海圖依節點數顯示可勾選按鈕（A、B、C...），點擊切換已探索狀態，並顯示目前進度（已選/總數），支援單張海圖「全選」「全清」；記錄會保存在本機
  • 船艇性能改為全自動計算：依「船艇等級」查表取得基礎值，加上四個部位配件等第的加成，即時算出探索性能／收集性能／巡航速度／航行距離／恩惠，等級或配件一有變動就重新計算
  • 移除「截圖辨識性能」OCR 功能與手動輸入性能欄位，性能欄位改為不可編輯的顯示徽章；船艇等級低於 51 級時，等級基礎值視為 0，只計算配件加成
  • 新增群組／標籤功能：可建立、重新命名、刪除自訂群組，OCR 辨識依目前群組範圍比對，新船自動歸入當前群組
  • 新增「手動新增船艇」功能：可直接輸入船名、等級、群組與回港時間
  • 新增單船「船艇設定」Modal：可設定四部位配件等級（船體／船尾／船首／艦橋），以及修改回港時間
  • OCR 重新辨識同一艘船時，保留原本手動設定的配件，不會被清空
  • 整體 UI 一致性調整：按鈕圓角、輸入框樣式、標題列元素垂直對齊統一；右側標題列寬度不足時自動換行，各按鈕寬度固定不縮放
